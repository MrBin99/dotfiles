# dotfiles

## Requirements

1. Ensure `Python` and `pip` are installed on the machine.
2. Install [Ansible](https://docs.ansible.com/) as global package :

```sh
sudo pip install ansible
```

## Install

Run installation using this command :

```
bash install.sh
```

## Uninstall

Uninstall using this command :

```
bash uninstall.sh
```
